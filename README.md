## waytopdf
a simple pyton script to convert all html pages to pdf in present working directory (not child)

### python version
python3

### How to use-
```$ python3 waytopdf.py
```
### requirements
1. pdfkit
you can use-
 ```$ pip install pdfkit```
may require- wkhtmltopdf
 ```$ sudo apt-get install wkhtmltopdf```
